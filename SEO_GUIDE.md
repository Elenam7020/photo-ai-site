# Гайд SEO Оптимизации

## Что уже сделано:

✅ **robots.txt** - файл для управления пауяли и индексации
✅ **sitemap.xml** - карта сайта для поисковых машин
✅ **Meta теги** - description, keywords, og:tags, canonical URL

## Что следует сделать мануально:

### 1. Регистрация в Google Search Console
- Перейти на https://search.google.com/search-console
- Кликнить "Начать"
- Внести вот: https://elenam7020.github.io/photo-ai-site/
- Подтвердить владение
- Угружая сить яво в раздел "Sitemap"

### 2. Регистрация в Yandex Webmaster
- Перейти на https://webmaster.yandex.ru/
- Добавить сайт
- Подтвердить владение
- Угружая sitemap.xml

### 3. Улучшение содержимого
- Оставить больше текста (мин 200-300 слов на странице)
- Написать блоги, со статьями
- По возможности купить власный домен

## Выполняются автоматически:
- Мобильная оптимизация (responsive design)
- Индексация роботами
- Open Graph теги для соцсетей
